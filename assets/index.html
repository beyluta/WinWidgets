Installed Widgets

Run on startup Launch WinWidgets automatically when you sign in
Hide in fullscreen Hide widgets when another app is in fullscreen mode
Restore last session Automatically reopen your widgets from the previous session

Default folder

Widgets are simple HTML files, embedded web content, displayed in a floating window.

To get started, open the default widgets folder. On Windows 11:

C:\\Users\{YOUR_USER}\Documents\WinWidgets

On supported Linux distributions, you'll find it under:

~/.local/share/WinWidgets

File structure

Each document of type HTML may contain CSS and JavaScript inside the same file. Drag and drop your HTML files to the default folder to load them into the application. Below is an example of how it might look like:

..\WinWidgets\ | |__ calculator.html | |__ weather.html | |__ clock.html

Layout and style

At the top of your HTML document add the following <meta /> tags (only the first 2 are required):

<!DOCTYPE html> <html> <head> <meta name="applicationTitle" value="myWidget" /> <meta name="windowSize" value="800 500" /> <meta name="windowLocation" value="300 300" /> <meta name="windowBorderRadius" value="500" /> <meta name="topMost" value="true" /> </head> </html>

Widgets support full transparency and are transparent by default, you may use CSS to set the background color:

<style> body { background-color: rgba(255, 255, 255, 0.5); } </style>
