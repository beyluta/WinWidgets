Installed Widgets

Run on startup Launch WinWidgets automatically when you sign in
Hide in fullscreen Hide widgets when another app is in fullscreen mode
Restore last session Automatically reopen your widgets from the previous session

Default folder

Widgets are simple HTML files, embedded web content, displayed in a floating window.

To get started, open the default widgets directory.

File structure

Each document of type HTML may contain CSS and JavaScript inside the same file. Drag and drop your HTML files to the default folder to load them into the application. Below is an example of how it might look like:

..\WinWidgets\ | |__ calculator.html | |__ weather.html | |__ clock.html

Layout and style

At the top of your HTML document add the following <meta /> tags (only the first 2 are required):

<!DOCTYPE html> <html> <head> <meta name="applicationTitle" content="myWidget" /> <meta name="windowSize" content="800 500" /> <meta name="windowLocation" content="300 300" /> <meta name="windowBorderRadius" content="500" /> <meta name="topMost" content="true" /> </head> </html>

Widgets support full transparency and are transparent by default, you may use CSS to set the background color:

<style> body { background-color: rgba(255, 255, 255, 0.5); } </style>

System functions

Use this to move the selected widget to a specific position

// Tell WinWidgets that we want to move the current selected widget window.chrome.webview.postMessage('MoveWindowToPosition(500, 500)');

Use this to get the global mouse position:

function GetMousePosition(x, y) { // Do something with x and y } // Tell WinWidgets that we want to get the global Position of the mouse window.chrome.webview.postMessage('GetMousePosition');

Use this to get the code of the current key pressed:

function GetCurrentKeyPressed(keyCode) { // Do something with the keyCode } // Tell WinWidgets that we want to get the global current key press window.chrome.webview.postMessage('GetCurrentKeyPressed');

Use these to control any active media stream such as video or audio

// Tell WinWidgets to stop/resume playback window.chrome.webview.postMessage('ToggleMediaPlayback'); // Tell WinWidgets to go to the next track window.chrome.webview.postMessage('NextMediaTrack'); // Tell WinWidgets to go to the previous track window.chrome.webview.postMessage('PreviousMediaTrack');
