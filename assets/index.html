Default folder

Widgets are simple HTML, CSS, and JavaScript files. Essentially just websites that are shown in a floating window. Start by navigating to the default WinWidgets folder under C:\\Documents\WinWidgets on Windows 11. On supported Linux distributions you can find the default directory under ~/.local/share/WinWidgets/.

File structure

Each .html file in the default folder needs a corresponding .cfg file with information about the widget. The name of both files must be the same except for the configuration file which must end with the .cfg suffix. Below is an example of the file structure: 
        WinWidgets/
        |__ calculator.html
        |__ calculator.html.cfg

Configuration file

The configuration file contains data about the current state of the widget. During the program's execution or at the end of execution, this file may be overwritten with new data. If no configuration file is present WinWidgets may create the file itself but it is highly recommended to create one yourself. This is an example of how it should look like: 
        title=My Widget
        width=600
        height=800
        x=512
        y=512
        title_bar=true
        top_most=false
        opacity=0.5
        radius=0

Widget controls

If title_bar=false in the configuration file then you may freely resize and move your widget by using the handle bar (top bar) of the widget window. If your widget does not have a handle bar, then you can no longer resize it, but you may still drag it by holding down the Windows-key + Left mouse button. Right-clicking the widget windows opens a context-menu where you can close the widget or set its top-most state.
